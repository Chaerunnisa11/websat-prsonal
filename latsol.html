Latihan Soal Matematika Kelas VI

Latihan Soal - Kurikulum Merdeka

Bilangan Bulat

  1. Apa hasil dari -3 + 5?
  2. Apa hasil dari -7 - 2?
  3. Tentukan hasil dari 6 * (-4)?
  4. Tentukan hasil dari -9 ÷ 3?
  5. Tentukan hasil dari 10 + (-3) - 4?

Himpunan

  1. Apakah 3 ∈ A, jika A = {1, 2, 3, 4}?
  2. Apakah 5 ∉ B, jika B = {2, 4, 6, 8}?
  3. Tentukan himpunan penyelesaian dari x ∈ Z, x ≥ 2 dan x ≤ 6
  4. Jika C = {1, 2, 3} dan D = {3, 4, 5}, apa hasil dari C ∩ D?
  5. Tentukan himpunan bagian dari A = {1, 2, 3}

Aljabar

  1. Selesaikan persamaan 2x + 3 = 7
  2. Tentukan nilai x jika 3x - 5 = 10
  3. Selesaikan persamaan 4x + 2 = 14
  4. Tentukan nilai x jika 5x + 3 = 23
  5. Tentukan nilai x jika x/2 + 3 = 8
