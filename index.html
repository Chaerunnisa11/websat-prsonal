Konsep Dasar Bilangan Bulat

Bilangan bulat adalah himpunan bilangan yang terdiri dari bilangan negatif, nol, dan bilangan positif. Himpunan bilangan bulat dilambangkan dengan ℤ, yang didefinisikan sebagai:

ℤ = { ..., -3, -2, -1, 0, 1, 2, 3, ... }

Jenis Bilangan Bulat

  • Bilangan Negatif: Bilangan yang lebih kecil dari nol (contoh: -1, -2, -3, dst.).
  • Bilangan Nol: Bilangan netral yang tidak positif atau negatif.
  • Bilangan Positif: Bilangan yang lebih besar dari nol (contoh: 1, 2, 3, dst.).

Operasi Dasar pada Bilangan Bulat

  • Penjumlahan:
    • Positif + Positif = Positif (contoh: 3 + 4 = 7).
    • Negatif + Negatif = Negatif (contoh: -2 + (-3) = -5).
    • Positif + Negatif = hasil tergantung bilangan yang lebih besar (contoh: 5 + (-3) = 2).
  • Pengurangan:
    • Sama dengan penjumlahan bilangan negatif (contoh: 7 - 3 = 4, 3 - (-2) = 5).
  • Perkalian:
    • Positif × Positif = Positif (contoh: 3 × 4 = 12).
    • Negatif × Negatif = Positif (contoh: -3 × (-4) = 12).
    • Positif × Negatif = Negatif (contoh: 3 × (-4) = -12).
  • Pembagian:
    • Aturan sama seperti perkalian (contoh: 12 ÷ (-3) = -4).

Sifat-Sifat Bilangan Bulat

  • Tertutup: Penjumlahan dan perkalian dua bilangan bulat selalu menghasilkan bilangan bulat.
  • Komutatif: Urutan bilangan tidak mempengaruhi hasil (contoh: a + b = b + a, a × b = b × a).
  • Asosiatif: Pengelompokan bilangan tidak mempengaruhi hasil (contoh: (a + b) + c = a + (b + c)).
  • Identitas:
    • Identitas Penjumlahan: 0 (contoh: a + 0 = a).
    • Identitas Perkalian: 1 (contoh: a × 1 = a).
